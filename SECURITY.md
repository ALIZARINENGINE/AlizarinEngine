# Security Policy

## Supported Versions

Only the latest stable release of the ALIZARIN Engine is supported with security updates.

| Version | Supported |
| :--- | :--- |
| Latest | ✅ |
| Beta / Pre-release | ❌ |

## Reporting a Vulnerability

The ALIZARIN Engine team and community take security vulnerabilities seriously. We appreciate your efforts to responsibly disclose your findings.

**Please do NOT post security vulnerabilities in the public GitHub issues.**

If you discover a security vulnerability, please send a private email to us at:
**alizarin.dev@gmail.com**

We will make every effort to respond to you within 48 hours to acknowledge your report.

### What to Include

Please include the following details in your report:
* A clear description of the vulnerability.
* The component it affects (e.g., the Public API, the "Streamer" App, the UTAU-TTS server).
* Steps to reproduce the vulnerability.
* Any proof-of-concept code, if available.

We are committed to working with you to resolve the issue and will give credit (if you wish) upon the release of a public advisory.
